# SystemAir App

SystemAir Fan Cost Calculator and Quotation App for ERPNext.

## Features

*   **SystemAir Quotation**: Calculate costs and generate quotations for SystemAir fans.
*   **Smart Shipping Distribution**: Automatically distributes shipping fees among items based on their basic price.
*   **Detailed Cost Breakdown**: Calculates CIF, Dry DDP Cost, and Total Price using configurable factors (Currency, Customs, VAT, Margin).

## Installation

1.  Get the app
    ```bench get-app https://github.com/onespaceerp/systemair_app```
2.  Install the app
    ```bench --site [site-name] install-app systemair_app```
